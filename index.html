Cara Download & Install

Ikuti langkah mudah berikut untuk menginstall ZCMH

1

Download APK File

Klik tombol download di bagian bawah halaman ini atau kunjungi GitHub Releases kami. File APK akan terdownload ke perangkat Android Anda.

2

Aktifkan "Install Unknown Apps"

Buka Settings ‚Üí Security ‚Üí Install unknown apps, lalu aktifkan untuk browser atau file manager yang Anda gunakan untuk download.

3

Install Aplikasi

Buka file APK yang sudah didownload dari folder Downloads. Tap "Install" dan tunggu proses instalasi selesai.

4

Buka & Nikmati

Setelah instalasi selesai, buka aplikasi ZCMH dan mulai perjalanan kesehatan mental Anda. Semua fitur sudah siap digunakan!

Pertanyaan yang Sering Diajukan

Temukan jawaban untuk pertanyaan Anda

Apakah ZCMH benar-benar gratis? ‚ñº
Ya, 100% gratis! Semua fitur (Gratitude Journal, Binaural Beats, Meal Reminder) dapat digunakan tanpa biaya. Untuk konsultasi dengan psikolog/hypnotherapist, biaya disesuaikan dengan layanan profesional yang Anda pilih.
Mengapa tidak tersedia di Play Store? ‚ñº
ZCMH saat ini masih dalam tahap research dan akan dipublikasikan di jurnal ilmiah Naskah Prima. Distribusi melalui APK memungkinkan kami memberikan akses gratis kepada publik tanpa biaya publikasi Play Store ($25).
Apakah APK ini aman untuk diinstall? ‚ñº
Sangat aman! APK di-host di GitHub Releases yang terpercaya. Source code akan tersedia secara public untuk transparansi. ZCMH tidak meminta permission yang mencurigakan dan tidak mengandung malware. Aplikasi hanya meminta izin notifikasi untuk meal reminder dan akses media untuk binaural beats.
Apakah data jurnal saya aman? ‚ñº
Ya! Semua data jurnal disimpan secara lokal di perangkat Anda menggunakan Room Database. ZCMH tidak mengirim data pribadi Anda ke server manapun. Data Anda sepenuhnya privat dan hanya dapat diakses oleh Anda.
Bagaimana cara update aplikasi? ‚ñº
Saat ini, update dilakukan secara manual. Kunjungi halaman ini atau GitHub Releases kami untuk download versi terbaru, lalu install seperti biasa. Kami sedang mengembangkan fitur auto-update checker untuk kemudahan Anda.
Apakah ZCMH bisa menggantikan terapi profesional? ‚ñº
TIDAK. ZCMH adalah aplikasi pendukung (adjunct therapy) yang membantu Anda dalam self-care dan kesehatan mental sehari-hari. Jika Anda mengalami kondisi mental health yang serius, sangat disarankan untuk berkonsultasi dengan profesional melalui fitur konsultasi kami atau langsung ke psikolog/psikiater.
Berapa lama saya harus menggunakan ZCMH untuk melihat hasil? ‚ñº
Berdasarkan penelitian, efek gratitude journaling dapat dirasakan dalam 1-2 minggu penggunaan konsisten (2-4x per minggu). Binaural beats memberikan efek relaksasi segera (15-30 menit), sedangkan meal reminder membantu stabilitas mood dalam beberapa hari. Konsistensi adalah kunci!
